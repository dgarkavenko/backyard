# System Mapping

This note translates performative moves into possible mechanics and content structures.

## Maintenance moves -> possible systems
- perimeter inspection routes
- visible wear states
- preventative repair windows
- component degradation
- checklist tools or station logs
- maintenance debt meters hidden behind system reliability

## Scarcity moves -> possible systems
- parts economy with low abundance
- explicit time windows between phases
- repair vs upgrade tradeoffs
- salvage as risky field labor
- component quality tiers or damaged parts

## Anticipation moves -> possible systems
- imperfect sensors
- camera placement and angle coverage
- ambiguous sound cues
- delayed or partial radio info
- breach behaviors that support pattern reading without becoming random

## Adaptive defense moves -> possible systems
- repositionable turrets
- modular traps
- power routing choices
- decoys or sensor devices
- contingency slots rather than fully permanent builds

## Emergency moves -> possible systems
- overlapping alarms
- manual overrides
- emergency patch actions
- subsystem shutdowns
- sector lockdown states
- cascading faults

## Postponed gratification moves -> possible systems
- turrets and traps visible from observation points
- strong feedback when early preparation pays off
- moments of partial calm that preserve tension
- aftermath readability showing why a setup succeeded

## Solitude and radio moves -> possible systems
- sparse but meaningful radio exchanges
- delayed answers
- neighboring node status updates
- environmental traces of past operators
- station rituals presented without excessive exposition

## Key design rule
Do not add systems just because they are realistic.
Add systems only if they support one or more of the core moves in [[12 - Core Moves]].

## Prototype checklist
A prototype is on target if players naturally:
- inspect before being forced to
- postpone something valuable
- distrust at least one obvious signal
- create a fallback plan
- experience one authored soft failure
- feel satisfaction when prep pays off later
